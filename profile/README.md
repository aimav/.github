Aimav Project
=============

AI, MetaOS, Apps, Vibed and combined into one.<br>
Web version: https://aimav.github.io


How Things are Made
===================

AI
  - Free models, local models (web)

MetaOS
  - For the web version, the browser itself is the OS
  - Aimav manages data of apps into a single access point

Apps
  - A selection of open-source apps are integrated in
  - Especially those are client-only apps

Vibing
  - AI era is here, vibing is for productivity

The one
  - Everything needed by users would be added in one launcher
  - Cross-platform
